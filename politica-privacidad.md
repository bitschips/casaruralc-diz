# Política de Privacidad

Este sitio web es una **demo ficticia** creada con fines educativos.

- No recopilamos datos personales mediante formularios internos.
- Si contactas por correo o WhatsApp, compartirás datos de forma voluntaria.
- No se ceden datos a terceros con fines comerciales.

Para cualquier duda sobre privacidad, contacta en: `info@casaruralcadizficticia.com`.
